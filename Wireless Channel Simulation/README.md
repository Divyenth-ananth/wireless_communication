# Wireless Channel Simulation

This project simulates wireless transmission over noisy channels (AWGN & Rayleigh Fading) using different modulation schemes.

## Features
- Supports BPSK, QPSK, 16-QAM
- Simulates AWGN and Rayleigh Fading
- Plots BER vs SNR curves for comparison

## How to Run
```bash
pip install -r requirements.txt
python channel_simulation.py
